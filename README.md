# Advanced Machine Learning Labs

## Overview
This repository contains a series of advanced laboratory works completed as part of coursework in Advanced Machine Learning. The labs focus on modern learning paradigms that extend beyond classical supervised learning, including reinforcement learning, deep reinforcement learning, continual learning, data stream mining, and transfer learning.

## Repository Structure

- **Lab01 – Reinforcement Learning (RL)**  
  Introduction to reinforcement learning, including Markov Decision Processes, policies, and value-based methods.

- **Lab02 – Deep Reinforcement Learning (Deep RL)**  
  Extension of RL using neural network for high-dimensional state spaces.

- **Lab03 – Continual Learning**  
  Learning under non-stationary data distributions while mitigating catastrophic forgetting.

- **Lab04 – Data Stream Learning**  
  Handling continuous, potentially infinite data streams with concept drift adaptation.

- **Lab05 – Transfer Learning**  
  Reusing knowledge from one domain or task to improve learning in another.
